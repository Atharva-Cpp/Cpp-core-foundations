# Cisco C++ Essentials (Associate Level)

This repository contains my progress, lab exercises, and logic implementations as I complete the **C++ Certified Associate Programmer (CPA)** certification track. 

## 🛠️ Concepts Covered
* **Module 1:** Data Types, Variables, and Basic I/O (`cin`, `cout`)
* **Module 2:** Flow Control, Conditional Logic (`if`/`else`), and Switch statements
* **Module 3:** Loops and Iteration (`while`, `do-while`, `for`)
* **Module 4:** Logic Operators, Bitwise Operations, and Data Aggregates (Arrays/Vectors)
* **Module 5:** Functions (Scope, Parameters, Overloading) and Introduction to Pointers

## 📂 Repository Structure
* `/Labs` - Official exercises and assignments from the Cisco curriculum.
* `/Mini-Projects` - Functional, text-based terminal applications built to test architectural logic.
